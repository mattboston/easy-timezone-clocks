# Easy Timezone Clocks

A macOS menu bar app for tracking multiple time zones at a glance.

- **Website:** [shields.tv/easy-timezone-clocks](https://shields.tv/easy-timezone-clocks/)
- **iOS App:** [Easy Timezone Clocks on the App Store](https://apps.apple.com/app/easy-timezone-clocks/id6778570535)

## Bugs & Support

If you run into a problem or bug, please [open an Issue](../../issues/new) in this repository with a description of what happened and steps to reproduce it.
